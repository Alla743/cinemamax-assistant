# Cinemax Store — تطبيق سطح المكتب (ويندوز)

هذا المجلد يحتوي على غلاف Electron الذي يحوّل تطبيق الويب إلى تطبيق ويندوز أصلي
مع **الكشف التلقائي الفوري للفلاشات** عبر مكتبتي `usb-detection` و `drivelist`.

---

## 🚀 الطريقة الموصى بها — البناء التلقائي عبر GitHub Actions

أنت لا تحتاج جهاز ويندوز ولا تثبيت أي شيء. يكفي:

1. **اربط مشروع Lovable بـ GitHub** (من زر الـ `+` في الأعلى ← GitHub ← Connect).
2. ادفع أي تعديل (أو افتح Actions يدوياً من تبويب **Actions** ← `Build Windows Installer (.exe)` ← **Run workflow**).
3. انتظر ~5 دقائق. سيتم بناء `Cinemax-Store-Setup-1.0.0.exe` تلقائياً على خادم ويندوز سحابي.
4. حمّل الملف من تبويب **Actions** ← آخر تشغيل ← **Artifacts** ← `Cinemax-Store-Windows-Installer`.
5. نفّذه على جهاز المحل — التثبيت بضغطة واحدة، ويعمل بدون إنترنت.

### إصدار رسمي (Release)

لو ضغطت tag باسم يبدأ بـ `v` (مثل `v1.0.0`)، سيُنشَأ تلقائياً GitHub Release
ويُرفع فيه ملف الـ `.exe` ليحمّله أي شخص مباشرة.

---

## 🛠️ البناء يدوياً على جهاز ويندوز (اختياري)

تحتاج: Node.js 20+ مثبت على ويندوز.

```bash
# في جذر المشروع
npm install
npm run build

# ثم
cd electron
npm install
npm run dist:win
```

سيظهر ملف التثبيت في مجلد `release/`.

---

## ✨ كيف يعمل الكشف التلقائي؟

- `usb-detection` تستمع لأحداث توصيل/فصل أجهزة USB من نظام ويندوز.
- عند وقوع حدث، يستدعي `drivelist` لجلب قائمة الأقراص القابلة للإزالة + نقاط التركيب (drive letters).
- يقارن البرنامج القائمة الجديدة بالقديمة ويرسل `drive:attached` / `drive:detached` إلى الواجهة.
- الواجهة تستلم الحدث، تستدعي `scanDrive` (مسح متعمّق لكل الملفات)، وتعرض الحجم والسعر فوراً.

---

## 🐛 استكشاف الأخطاء

| المشكلة | الحل |
|---|---|
| `.exe` لا يعمل بدون إنترنت | تأكد أنك ركّبت من ملف الـ Setup (وليس فتحت ملف من المجلد المؤقت) |
| الفلاشة لا تُكتشف | افتح "إدارة الأقراص" في ويندوز وتحقق أن الفلاشة لها حرف drive |
| رسالة "drivelist not loaded" | لم يكتمل `npm install` داخل مجلد `electron/`. أعد تشغيله |
